If the part has this at the end of its name:
- <code> _1 </code> -> print 1

- <code> _a </code> -> print with your accent color

When installing, make sure the idlers are at right height to keep belts parallel.

	- No need for support. 
	- 3-4 Perimeters
	- 30-40% Infill
